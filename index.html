iQuery

兼容 IE 8+ 的轻量级、增强型 jQuery 兼容 API！

立即获取 GitHubTM Support

上手快

  • 实现了所有jQuery 常用 API
  • 扩展 API完全遵循 jQuery API 命名风格、传参形式

兼容佳

  • 最低IE 8的主流浏览器支持
  • 兼容模块以ECMA、W3C 标准 API的形式开放
  • 中小规模复杂度jQuery 插件无缝切换

适用广

  • 体积、兼容范围 适中，一库通用、缓存率利用高
  • Polyfill、Plugin 提供 API 独立包， 快速适配原有jQuery 兼容项目

进阶易

  • iQuery+内建的 通用视图接口大幅简化 jQuery 插件开发
  • 轻松实现组件化开发、面向对象式的解耦/通信
  • （兼容组件库实例 —— EasyWebUI）
